Selected Projects & Papers

Medical Report Generation Pipeline for a National Key R&D Program Subtask

Du Bo's Group, Wuhan University · National Key Research and Development Program of China (2023YFC2705700)

Built the end-to-end pipeline for a medical report generation subtask, including dataset construction, model implementation, and training workflow design.

Completed dataset preparation, training pipeline development, and iterative experiments, producing reusable engineering workflows and stage-wise research results. Although the subtask was not carried through to final deployment because of project-level scheduling changes, the methods and engineering pipeline have been retained for reuse within the group.

FedProCL: Prototype-Guided Contrastive Federated Learning for Medical Image Classification under Concept Shift

First Author · BIBM (CCF-B)

Proposed a prototype-guided contrastive learning framework that uses class prototypes as stable semantic anchors to align multi-center feature distributions, improving cross-center generalization and classification robustness without data sharing.

MedMerge: A Training-Free Model Merging Framework for Medical Knowledge Transfer into Vision-Language Models

First Author · TRPMS (JCR Q1)

Proposed a training-free model merging method that transfers knowledge from multiple source models into a vision-language model without retraining or access to the original data, which is well suited to privacy-constrained medical collaboration scenarios.

Rethinking Federated Prompt Learning for Medical Images: From Textual Tuning to Visual Manifold Anchoring

First Author · ICML (CCF-A, under review)

Proposed a visual manifold anchoring paradigm that uses LLM-derived semantic topology priors to regularize client prompts, improving stability and transferability across devices, protocols, and distribution shifts.

Class Similarity-Based Multimodal Classification under Heterogeneous Category Sets

Co-First Author · Pattern Recognition (JCR Q1, under review)

Modeled class similarity to address label misalignment across heterogeneous data sources, enabling cross-source generalization without forcing a unified label space.

Injecting Reasoning into Vision-Language Models via Weight-Decomposed Merging

Second Author · Neural Networks (JCR Q1, under review)

Proposed a weight-decomposed merging strategy to inject reasoning ability into vision-language models in a controllable way, reducing interference and catastrophic forgetting while preserving both base capabilities and reasoning performance.
